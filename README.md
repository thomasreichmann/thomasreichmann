**Things I built because someone needed them**

- [**Nexus**](https://nexus.thomasar.dev) — my mom is a photographer with a decade of shoots on a closet full of external drives. One started failing, and a corruption scare nearly took a wedding with it. Cold storage is the right home for that data, cheap and durable, but consumer tools charge you for instant access you never use. So I built the archival-first version on S3 Glacier: store cold, pay cold, wait when you actually need a file back. Solo, end to end. ([code](https://github.com/thomasreichmann/nexus))
- [**thomasar-cv**](https://cv.thomasar.dev) — keeping my own résumé in a word processor turned every edit into manual layout surgery, so I built a tool that treats a résumé as structured data with an ATS-safe PDF export. Every real decision is written up as an [ADR](https://github.com/thomasreichmann/thomasar-cv/tree/main/docs/decisions), because I build side projects the way things that have to last get built. ([code](https://github.com/thomasreichmann/thomasar-cv))
